== BlueXPの優れた運用耐障害性

* BlueXPの運用耐障害性に関するURLは何ですか？*

URLの場合は、ブラウザで次のように入力します。 https://console.bluexp.netapp.com/["https://console.bluexp.netapp.com/"^]



== ライセンス

* BlueXPの運用耐障害性を使用するにはライセンスが必要ですか？*

ネットアップライセンスファイル（NLF）は必要ありません。
